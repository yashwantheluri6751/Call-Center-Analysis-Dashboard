# Call Center Analysis Dashboard

This project analyzes call center operational data using Power BI to evaluate agent performance, response time, abandonment rate, and customer service efficiency.

## Tools Used
- Power BI
- Excel/CSV Dataset
- DAX & Power Query

## Key Insights
- Agent performance tracking
- Customer response trend analysis
- KPI dashboard visualization

## Purpose
To demonstrate data visualization, business insight generation, and dashboard development skills.
